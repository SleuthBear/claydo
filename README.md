# claydo
## Clay Done in Odin
This is a rewrite of the Clay (C-Layout) library in Odin.

This is NOT the official Odin bindings for Clay, this is an entire re-write. If you want to use the bindings, check the Clay repo, they are in the bindings folder.

Credit to Nic Barker for the original implementation in C https://github.com/nicbarker/clay/tree/main
